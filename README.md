# Blooket Hacks

A collection of JavaScript hacks and utilities for Blooket, an interactive gaming platform for classrooms.

## Overview

This repository contains various hacks and enhancements for Blooket to improve gameplay and user experience.

## Features

- Enhanced gameplay mechanics
- Utility functions
- Performance improvements

## Getting Started

### Installation

1. Clone this repository
2. Review the available hacks in the `/hacks` directory
3. Follow individual hack documentation for implementation

## Usage

Include the hack scripts in your Blooket session or use them as browser console scripts.
