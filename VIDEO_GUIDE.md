## 📋 Quick Reference for Video Creators

### Camera/Screen Recording Tips:
- Use OBS Studio or ScreenFlow for smooth recording
- Zoom in on console text so it's readable
- Show mouse cursor movements clearly
- Use cursor highlighter for emphasis
- Record at 1080p or higher

### Audio Tips:
- Speak clearly and slowly
- Use background music (royalty-free)
- Add sound effects for clicks/actions
- Normalize audio levels

### Editing:
- Add titles/subtitles for each section
- Highlight important text with arrows
- Use zoom-in effects for small details
- Include code snippets as text overlays
- Add end screen with GitHub link

### Thumbnails:
- Show Blooket game interface
- Add "HACKS" in big text
- Include device emoji (📱, 💻, etc.)
- Make it colorful and eye-catching
